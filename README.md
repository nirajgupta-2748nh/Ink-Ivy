## Overview

Ink & Ivy is a front-end web project that represents a modern online bookstore interface. It allows users to explore featured books, browse best sellers, and interact with a clean and visually appealing layout.

This project was developed collaboratively as a practice exercise to strengthen core web development concepts such as layout design, styling, and responsive structuring.

---

## Features

* Responsive navigation bar with dropdown menu
* Hero section with call-to-action
* Book display using card-based layout
* Sections for Featured Books and Best Sellers
* Hover animations for better user interaction
* Contact section with form
* Clean and structured UI design

---

## Tech Stack

* HTML5
* CSS3
* Bootstrap
* Font Awesome
* Google Fonts (Material Symbols)

---

## Key Implementation Details

* Layout structured using CSS Grid and Flexbox 
* Interactive card hover effects for improved UI experience 
* Dropdown navigation implemented using pure CSS 
* Semantic HTML used for better readability and structure
